# cal_ipo_gain
计算季度新股收益情况，目前最稳定版本为新股账户开在华泰证券的收益统计

## 主要功能

### 计算收益统计

- [x] 华泰，`cal_ht.py`
- [ ] 中信
- [ ] 财通

### 对比收益统计

使得文件夹A中所有的收益统计与文件夹B中出现的收益统计进行比对，如有异常则返回，`check_gain.py`

### 支持收益计算类型

- [x] 传统新股
- [x] 可转债
- [x] 注册制创业板，含限售股解禁
- [x] 科创板
- [ ] 新三板

## 注意

1. 计算的日期区间是指卖出日在区间内的ipo收益计算
2. 原始交割单需要向前多传入一个月的数据
3. `ipo.csv` 是通过Tushare进行的发行价请求，以减少wind请求的依赖
